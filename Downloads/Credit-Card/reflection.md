# Reflection

## Did it help you move faster?

Yes, the coding assistant Claude helped me move faster during the development of my project. It provided suggestions for optimizing the machine learning model and improving its performance. For example, it recommended techniques such as using SMOTE to handle class imbalance and applying hyperparameter tuning to improve the model's F1-score. These suggestions helped me quickly implement improvements and experiment with better configurations for the fraud detection model.

## Did it generate incorrect or surprising suggestions?

At times, the assistant generated large sets of code modifications at once, which was somewhat surprising. While many suggestions were useful, it required careful review to determine which changes should actually be implemented so that the model's performance improved instead of degrading. Additionally, using techniques like Stratified K-Fold cross-validation significantly increased the training time, which was an unexpected practical challenge during experimentation.

## Where was it most or least useful?

The assistant was most useful in validating the machine learning workflow and ensuring that the model was trained using appropriate techniques for real-world datasets, such as handling class imbalance and preventing overfitting. It helped verify that the evaluation metrics and preprocessing steps were correct for a fraud detection task. However, it was less helpful when implementing automation tasks such as versioning or artifact logging in the training pipeline, which required more manual setup and configuration.
